# MVP 规格

## 目标用户

- 需要频繁登录 Linux 主机的开发和运维
- 需要统一连接 Windows 和 Linux 的内部工具使用者
- 希望比纯终端更易管理连接、命令和日志的团队

## 首版必须支持

### 1. 主机管理

- 新增主机
- 编辑主机
- 删除主机
- 主机分组
- 支持以下字段：
  - 名称
  - 地址
  - 端口
  - 用户名
  - 认证方式
  - 默认 shell
  - 平台类型

### 2. 远程终端

- 多标签
- 连接状态提示
- 自动滚动
- 复制粘贴
- 窗口 resize 同步

### 3. 平台兼容

- Linux：SSH
- Windows：SSH

说明：

- 首版不强制支持 WinRM
- 对 Windows 机器要求启用 OpenSSH Server

### 4. 命令辅助

- 最近执行命令
- 收藏命令
- 常用命令分类

### 5. 基础文件传输

- 上传单文件
- 下载单文件
- 首版只基于 SFTP

## 首版不做

- 远程桌面
- Docker / Kubernetes 集成
- 批量任务编排
- 审批流
- 多用户协作
- 插件系统

## 关键交互

### 连接主机

1. 从左侧主机列表选择目标
2. 点击“连接”
3. 新建标签页
4. 显示连接中状态
5. 成功后进入交互 shell

### 执行收藏命令

1. 打开命令面板
2. 选择已收藏命令
3. 插入当前终端
4. 二次确认后执行

## 非功能要求

- 冷启动时间：尽量小于 2 秒
- 单会话输入延迟：局域网内尽量接近原生 SSH
- 崩溃恢复：异常退出后保留主机配置
- 安装复杂度：尽量免安装或单步安装

## 发布形式

### Windows

- 首选：单文件 `.exe`
- 可选：轻量安装器

### Linux

- 首选：单文件可执行程序
- 可选：`.tar.gz`

## 后续增强

- WinRM 支持
- 主机标签和搜索
- 批量连接
- 审计日志
- 双栏文件管理器
